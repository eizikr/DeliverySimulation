# DeliverySimulation

This project simulates a shipping company.

Displays tracking of truck location and condition, packages to be shipped, and tracking of shipping details.
The project is written in Java and includes Threads, GUI, design pattern and more

Some details:
Source of packages are the red circles on top and destination of packages are the red circles on bottom.
Vans (Blue) takes packages from Source to branch and from branch to destination.
Standard trucks (Green) takes packages from branches to hub and from hub to branches.
Non standard trucks (Red) takes Non standard packages from source to destination.

User manual:
1. Run the main program
2. Click on "Create system"
3. Enter the branches, tracks and packages numbers and click ok
4. Click "Start" to run the program
5. To freez the program you can click on "Stop" button, and to continue click "Resume".
6. To watch packages tracking click "All packages info"
7. To watch specific branch info click "Branch info"
